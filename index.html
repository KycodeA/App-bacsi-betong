Nhóm 4: Xác Định Tỉ Lệ % Diện Tích Mặt Đường Bị Nứt

Ứng dụng xử lý ảnh từ clip mặt đường BTXM (10 đoạn x 500m) để tính toán diện tích nứt lưới và nứt đơn theo từng phân đoạn 50m.

Hạn nộp: Thứ 2, 22/06/2026

Nguyên Tắc & Phương Pháp Đánh Giá

Vết nứt hình lưới: Nhận biết bằng mô hình, tiến hành khoanh vùng khu vực nứt và tính toán trực tiếp diện tích vùng bao tổn thương.
Vết nứt đơn (Ngang, dọc, chéo): Loại bỏ hoàn toàn các khe nối cố định của mặt đường BTXM. Diện tích được chuẩn hóa: S = Chiều dài vết nứt x 0.3m.

Môi Trường & Thư Viện Yêu Cầu (Requirements)

Hệ thống triển khai trên nền tảng Python 3.10+. Môi trường có thể thiết lập nhanh chóng qua file requirements.txt:

# Cú pháp cài đặt nhanh trên Terminal / Command Prompt:

pip install -r requirements.txt

# Chi tiết các thư viện cốt lõi cấu hình trong file:

opencv-python>=4.8.0 # Xử lý trích xuất frame hình và bẻ phối cảnh góc quay

numpy>=1.24.0 # Tính toán ma trận diện tích hình học thực tế

ultralytics>=8.1.0 # Khởi chạy mô hình học sâu YOLOv8 nhận diện vết nứt

pandas>=2.0.0 # Thống kê và cấu trúc dữ liệu đầu ra 100 phân đoạn

Yêu cầu 1: Đoạn Code Xác Định Tỉ Lệ %

AntiGravity / Python

Cấu trúc mã nguồn thực hiện chu trình: Cắt ảnh từ video clip, bẻ phối cảnh (Perspective Transform) chuẩn hóa góc quay camera, chạy mô hình và tính tỷ lệ tổn thất trên diện tích thực tế. 

# Quy trình xử lý tự động: Cắt ảnh -> Bẻ phối cảnh -> Dự đoán -> Tính % diện tích
import cv2
import numpy as np

def process_road_video(video_path):
    cap = cv2.VideoCapture(video_path)
    # 1. Trích xuất frame hình và bẻ phối cảnh để xử lý góc quay từ camera
    # 2. Chạy mô hình (YOLOv8 / UNet) đã được huấn luyện sẵn
    # 3. Phân tách nứt lưới và nứt đơn (Bỏ qua khe nối cố định)
    
    ROAD_WIDTH, SEGMENT_LENGTH = 3.5, 50.0  # Kích thước phân đoạn (mét)
    total_road_area = ROAD_WIDTH * SEGMENT_LENGTH
    
    # Tính toán diện tích nứt thực tế
    crack_mesh_area = get_mesh_area()
    crack_single_length = get_single_crack_length()
    crack_single_area = crack_single_length * 0.3 
    
    total_crack_area = crack_mesh_area + crack_single_area
    percentage = (total_crack_area / total_road_area) * 100
    
    return min(round(percentage, 2), 100.0)

Yêu cầu 2: Kết Quả Tỉ Lệ % Nứt Theo Từng Phân Đoạn

Tổng cộng 100 số liệu thu thập từ 10 đoạn mặt đường (mỗi đoạn chia làm 10 phân đoạn 50m).

Đoạn đường 0-50m 50-100m 100-150m 150-200m 200-250m 250-300m 300-350m 350-400m 400-450m 450-500m
